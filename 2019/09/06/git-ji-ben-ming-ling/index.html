git基本命令
git
发布日期:   2019-09-06
文章字数:   303
阅读时长:   1 分
阅读次数:  

git init

初始化本地仓库

git config

配置git全局信息
$ git config --global user.name 'name'
$ git config --global user.email 'email'

git add

$ git add <filename>
添加指定文件：
$ git add .
添加目录下所有文件

git rm

$ git rm --cached <filename>
删除添加的指定文件

git status

查看添加的文件队列

git commit

默认会弹出编辑日志框
$ git commit -m 'log'
直接指定日志内容提交

git remote

$ git remote add <SSH URO or HTTP URL>
$ git remote add origin remoteUrl
添加远程仓库
$ git remote -v
查看绑定的远程仓库
$ git remote remove <remoteName>
$ git remote remove origin

git push

$ git push -u origin master
推送项目到远程仓库

git pull

从远程仓库拉取项目（同步）
$ git pull origin master --allow-unrelated-histories
将本地仓库和远程仓库进行关联

git clone

从远程仓库拉取项目（从无到有）

git branch

$ git branch test
创建分支：test

git checkout

$ git checkout test
切换到test分支

git merge

$ git merge test
合并test分支内容到主干
